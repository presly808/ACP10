
1) Download git https://git-scm.com/downloads 
2) Install git 
For windows users
3) run git bush = Start->Search Git Bash -> run
4) use command  'cd' to change location
		'ls' show folder content
		'pwd' show current location path
		
5) go to location where project will be cloned
6) write command 'git clone <repository-url>'
   example       'git clone https://github.com/presly808/ACO8.git'

SET UP IDEA 

1) create new idea project -> set right location for idea project(cloned project directory)

2) tell idea about maven
	- open global setting 'Ctrl+Alt+S'
	- item 'Vesion Control' -> Git
	- check 'Git executable path'
	- press test for verify

3) then activate git in IDEA
	- VCS -> Enable Version Control Integration -> Choose Git -> Press Ok

4) After above steps

	use button 'update' in IDEA or shortcut 'Ctrl+T'
	if you have rights, you can push your changes to server via 'Ctrl+K'
